Automatic theming in Linux

I always thought automatic theming is impossible in WM and impossible to do correctly in DE (I have no experience with DE). But apparently this is the easiest thing in the world. Lemme explain

  • Download themix (https://github.com/themix-project/themix-gui)
  • Select/create theme and export in ~/.themes
  • Read this https://wiki.archlinux.org/title/Dark_mode_switching
  • Use nwg-look to test if theme applies (DONE if you dont want automatic)
    • Note that a theme can only be dark mode or light mode despite having a setting called prefer dark/light
    • So keep note of light themes (your UI and firefox/brave should be white) and dark respectively
  • Use gsettings to set theme (requires theme name) from terminal
  • Make a script
  • Profit
#!/usr/bin/env sh

dark_theme="oomox-lcars_scrot-dark"
light_theme="oomox-chesnut-light"

theme_status="$(gsettings get org.gnome.desktop.interface color-scheme)"

if [ $theme_status = "'prefer-light'" ]
then
  notify-send "dark theme set up"
  gsettings set org.gnome.desktop.interface gtk-theme $dark_theme
  gsettings set org.gnome.desktop.interface color-scheme "prefer-dark"
else
  notify-send "light theme set up"
  gsettings set org.gnome.desktop.interface gtk-theme $light_theme
  gsettings set org.gnome.desktop.interface color-scheme "prefer-light"
fi
