modern web is so fast

Aagaman Luitel | words: 120 | 0 minutes | 2024.10.10

I was doing some react dev and I saw this today

  • https://www.youtube.com/watch?v=-Ln-8QM8KhQ

My localhost barebones react/daisyui website clearly takes 200ms to load and this old framework agressive loads content before the browser changes its url. Changing the web to less bloat would be so better. Not having a rgb-drop-shadowed-on-scroll-change-effect horizontal box css in your website styled with 200LOC of css is much more healthier than having one. Modern standards are bad, it should be changed. And on JS side, most JS framework do the same thing but uses different config/init and design concepts which we like over other and I think adapting to one even if we don’t like is better than creating a new one over and over again.

RTFM is the true path of every developer. But it would happen only if there's no way out.
